# 《星舰幸存者》项目简报

## 正式名称

《星舰幸存者》（Starship Survivors）

## 项目定位

本项目是一个无依赖浏览器 Canvas 原型，方向固定为：

- 无限宇宙生存
- 飞船割草
- Roguelike 局内成长
- 星球势力挑战
- 周期性休息站中转

玩家驾驶飞船在深空中持续生存、移动、选择路线、积累局内 Build，并自主决定是否攻打星球势力或前往休息站恢复与升级。

## 核心目标

- 存活时间越久越好。
- 飞船不断进化，从 T1 探索船成长到后期高阶舰体。
- 玩家自由选择探索路线与挑战节奏。
- 玩家可自由选择是否攻打星球。
- 每局通过武器、资源、星球奖励与休息站选择形成不同 Build。

## 美术风格

资产统一遵守：深空背景、简化卡通科幻、圆润轮廓、高识别度、低复杂度、轻量 2.5D、整体干净统一，适合原型快速接入。

明确不追求厚重写实、不做复杂写实插画风、不加入难以在小尺寸读取的细碎结构。

## 当前阶段

本阶段优先完成“游戏资产自动生产系统”和“资产标准化目录结构”：

1. 建立稳定的 `assets/` 分层目录。
2. 建立全量 manifest，给每个资产分配 ID、分类、用途、prompt、尺寸、状态和路径。
3. 提供 OpenAI Image API 批处理脚本、contact sheet、导出、失败重试、完整性检查和报告生成能力。
4. 为后续 Canvas 原型接入图片资产预留轻量路径配置，不引入复杂前端框架。
