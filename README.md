# Space Game

一个无依赖的浏览器端 2D 太空策略射击游戏 Canvas 原型。玩家驾驶飞船在太空地图中移动，观察多个星球的攻击范围，并选择是否进入危险区域。

当前版本是第一阶段最小可玩原型，重点是确保项目不依赖 npm、Vite、TypeScript 或 Phaser，也能直接在浏览器中运行。

## 技术方案

- 原生 HTML
- 原生 CSS
- 原生 JavaScript
- 浏览器 Canvas 2D API
- 无外部 CDN
- 无 npm 依赖

## 运行方式

无需安装依赖，也不需要启动开发服务器。

方式一：直接双击打开：

```text
index.html
```

方式二：在浏览器中选择“打开文件”，然后打开项目根目录下的 `index.html`。

## 当前功能

- 黑色太空背景。
- 随机星星背景。
- 玩家飞船临时三角形图形。
- WASD / 方向键移动。
- 空格发射子弹。
- 3 个不同颜色的星球占位圆形。
- 每个星球显示攻击范围圆圈。
- 玩家进入星球攻击范围时显示警告提示。
- 右上角显示简易小地图。
- 左上角显示玩家生命值和当前状态。

## 项目结构

```text
index.html          页面入口，可直接用浏览器打开
src/main.js         Canvas 游戏主逻辑
src/styles.css      页面与画布样式
docs/               游戏策划、开发计划和资产清单
```

## 后续计划

1. 在无依赖 Canvas 原型中继续验证基础战斗，例如星球生命值、子弹命中和玩家受伤。
2. 当玩法方向稳定后，可以再迁移到 Phaser 或其他游戏框架。
3. 迁移到 Phaser 时再恢复 npm 项目结构、构建流程和 TypeScript 类型检查。
4. 后续逐步加入休息站、小地图扩展、武器升级、陨石、跟踪机关、实体撞击和破散弹幕。
